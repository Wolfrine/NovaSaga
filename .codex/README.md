# NovaSaga Codex Local Prompts

This directory holds repo-local Codex prompt wrappers.

Intended slash prompt:

```text
/novasaga-curate
```

If the current Codex client does not display repo-local `.codex/prompts` entries, use the skill directly:

```text
/use novasaga-lore-curator
```

or ask:

```text
Use the novasaga-lore-curator skill and curate new raw notes into the active wiki.
```

The canonical behavior remains in:

```text
.agents/skills/novasaga-lore-curator/SKILL.md
```
