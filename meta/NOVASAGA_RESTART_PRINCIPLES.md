# NovaSaga Restart Principles

## Purpose

NovaSaga is restarting with a clean active structure while preserving legacy context in archive.

## Principles

- The active wiki should be compact and useful for reasoning.
- Archive material is evidence, not automatic canon.
- The story should grow through durable pages, not scattered prompt outputs.
- Uncertainty should remain visible until the user resolves it.
- Worldbuilding should serve story, character, and theme.
- Prose drafts should be protected from unrequested rewrites.
- Future agents should be able to open the repo and understand the current direction without chat history.
